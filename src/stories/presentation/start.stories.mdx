# Что такое Storybook ?


### экземпэл истории

```
export default {
  title: 'Example/Button',
  component: Button,
  argTypes: {
    backgroundColor: { control: 'color' },
  },
}
const Template: ComponentStory<typeof Button> = (args) => <Button {...args} />;

export const Primary = Template.bind({});
Primary.args = {
  primary: true,
  label: 'Button',
};

```

**Зная этот каркас мы можем делать различные  тесты компонент и смотреть как они себя ведут в различных
ситуациях**
